# モバイルデバイスからアクセスする方法

## 方法1: 開発サーバーをネットワーク公開（推奨）

### 手順

1. **開発サーバーを停止**
   - 現在動いているPowerShellウィンドウを閉じる

2. **ネットワーク公開モードで起動**
   ```powershell
   cd "C:\Users\sqjyp\OneDrive\デスクトップ\management\tavi-ct-input"
   npm run dev:network
   ```

3. **PCのIPアドレスを確認**
   - PowerShellで以下を実行：
   ```powershell
   ipconfig
   ```
   - 「IPv4 アドレス」を確認（例: `192.168.1.100`）

4. **モバイルデバイスからアクセス**
   - モバイルデバイスを**同じWi-Fiネットワーク**に接続
   - モバイルのブラウザで以下にアクセス：
     ```
     http://[PCのIPアドレス]:3000
     ```
   - 例: `http://192.168.1.100:3000`

### 注意事項

- PCとモバイルデバイスが**同じWi-Fiネットワーク**に接続されている必要があります
- ファイアウォールの警告が出た場合は、「アクセスを許可」を選択してください
- セキュリティ上、開発中のみ使用してください

## 方法2: 本番ビルドしてデプロイ

### 静的サイトとしてビルド

1. **ビルド**
   ```powershell
   npm run build
   ```

2. **静的エクスポート**
   - `next.config.mjs`を編集して`output: 'export'`を設定
   - ビルド後、`out`フォルダに静的ファイルが生成されます

3. **デプロイ**
   - GitHub Pages、Netlify、Vercelなどにデプロイ
   - または、Webサーバーにアップロード

## トラブルシューティング

### モバイルから接続できない場合

1. **ファイアウォールの確認**
   - Windowsファイアウォールでポート3000を許可

2. **IPアドレスの確認**
   - PCのIPアドレスが正しいか確認
   - `ipconfig`で再確認

3. **ネットワークの確認**
   - PCとモバイルが同じWi-Fiに接続されているか確認

4. **ブラウザの確認**
   - モバイルのブラウザで `http://` を忘れずに（`https://` ではない）
